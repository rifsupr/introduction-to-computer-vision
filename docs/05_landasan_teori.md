# Landasan Teori

# 1. Artificial Intelligence

Artificial Intelligence (AI) adalah teknologi yang memungkinkan komputer melakukan tugas yang biasanya membutuhkan kecerdasan manusia.

Contoh implementasi AI:
- Pengenalan wajah
- Chatbot
- Object detection
- Speech recognition
- Recommendation system

# 2. Deep Learning

Deep Learning merupakan cabang Machine Learning yang menggunakan Neural Network dengan banyak layer.

Deep Learning sangat efektif untuk:
- Pengolahan gambar
- Video
- Suara
- Dan bahasa alami

# 3. Convolutional Neural Network (CNN)

CNN merupakan algoritma Deep Learning yang dirancang khusus untuk pengolahan gambar.

CNN bekerja dengan:
- Mengenali pola visual
- Mengekstraksi fitur gambar
- Dan melakukan klasifikasi objek

CNN menjadi dasar utama pada:
- Image classification
- Object detection
- OCR
- Segmentation
- Dan face recognition

# 4. Computer Vision

Computer Vision merupakan bidang AI yang memungkinkan komputer memahami informasi visual dari gambar dan video.

Tujuan utama Computer Vision:
- Mendeteksi objek
- Mengenali objek
- Memahami isi gambar
- Dan melakukan analisis visual

# 5. Object Detection

Object Detection adalah teknik Computer Vision untuk:
- Mendeteksi lokasi objek
- Menentukan jenis objek
- Dan memberikan bounding box pada objek

Output utama object detection:
- Koordinat objek
- Class object
- Confidence score

# 6. Image Classification

Image Classification adalah proses mengenali isi utama suatu gambar.

Model akan memprediksi:
- Jenis objek
- Kategori gambar
- Dan probabilitas prediksi

# 7. YOLO (You Only Look Once)

YOLO adalah algoritma object detection berbasis CNN yang mampu mendeteksi objek secara realtime.

YOLO bekerja dengan:
- Membaca gambar satu kali
- Mendeteksi lokasi objek
- Dan menentukan class object secara bersamaan

# 8. YOLOv8

YOLOv8 merupakan versi terbaru dari YOLO yang dikembangkan oleh Ultralytics.

Keunggulan YOLOv8:
- Cepat
- Ringan
- Akurat
- Dan mudah digunakan

YOLOv8 mendukung:
- Object detection
- Classification
- Segmentation
- Dan pose estimation

# 9. OpenCV

OpenCV adalah library Computer Vision yang digunakan untuk:
- Membaca gambar
- Memproses citra
- Menampilkan visualisasi
- Dan manipulasi gambar

Pada project ini OpenCV digunakan untuk:
- Membaca gambar
- Membuat bounding box
- Dan menampilkan label object

# 10. Ultralytics

Ultralytics merupakan framework modern yang menyediakan implementasi YOLOv8.

Framework ini memudahkan:
- Loading model
- Inference
- Training
- Dan deployment model YOLO

# 11. Kesimpulan

CNN dan Computer Vision merupakan teknologi penting dalam Artificial Intelligence modern.

YOLOv8 menjadi salah satu algoritma populer karena mampu melakukan object detection dan classification dengan cepat dan akurat.

Melalui kombinasi YOLOv8 dan OpenCV, sistem dapat memahami isi gambar dan mengenali objek secara otomatis.
