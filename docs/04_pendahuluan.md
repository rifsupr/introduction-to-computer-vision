# Pendahuluan

# 1. Latar Belakang

Perkembangan Artificial Intelligence (AI) saat ini berkembang sangat pesat, terutama pada bidang Computer Vision dan Deep Learning.

Computer Vision memungkinkan komputer untuk memahami gambar maupun video secara otomatis seperti cara manusia melihat lingkungan sekitar.

Salah satu implementasi paling populer dalam Computer Vision adalah Object Detection, yaitu proses mendeteksi dan mengenali objek pada gambar maupun video.

Teknologi Object Detection banyak digunakan dalam:
- Sistem keamanan
- Kendaraan otomatis
- Smart city
- Monitoring lalu lintas
- Industri
- Retail
- Dan media sosial

Salah satu algoritma Object Detection modern yang populer adalah YOLO (You Only Look Once).

YOLO mampu melakukan deteksi objek secara realtime dengan performa yang cepat dan akurasi tinggi.

Pada project ini digunakan YOLOv8 sebagai implementasi Computer Vision untuk melakukan:
- Object detection,
- Object classification,
- Dan visualisasi hasil deteksi objek.

# 2. Rumusan Masalah

Berdasarkan latar belakang tersebut, rumusan masalah pada project ini yaitu:

1. Bagaimana implementasi Computer Vision menggunakan YOLOv8?
2. Bagaimana sistem mendeteksi objek secara otomatis?
3. Bagaimana melakukan klasifikasi objek menggunakan model Deep Learning?
4. Bagaimana visualisasi hasil object detection dilakukan?

# 3. Tujuan Project

Tujuan dari project ini yaitu:

- Memahami implementasi Computer Vision
- Mempelajari object detection menggunakan YOLOv8
- Mempelajari image classification
- Dan membangun sistem AI sederhana berbasis Deep Learning
