# Penutup

# 1. Kesimpulan

Berdasarkan hasil implementasi dan pengujian yang telah dilakukan, project “Object Detection & Classification menggunakan YOLOv8” berhasil dikembangkan menggunakan teknologi Computer Vision berbasis Deep Learning.

Sistem mampu:
- Mendeteksi objek pada gambar
- Melakukan image classification
- Menampilkan bounding box
- Menampilkan confidence score
- Dan menghasilkan top-5 prediction secara otomatis

Implementasi dilakukan menggunakan:
- YOLOv8 Detection
- YOLOv8 Classification
- Dan OpenCV sebagai pengolahan citra

Hasil pengujian menunjukkan bahwa:
- Sistem mampu mendeteksi objek dengan baik
- Classification model bekerja cukup akurat
- Dan visualisasi detection berjalan dengan baik

Pada beberapa pengujian, model detection mengalami kesalahan klasifikasi awal, namun model classification mampu memberikan hasil prediksi yang lebih akurat.

Hal ini menunjukkan bahwa kombinasi:
- Object detection
- Dan image classification
mampu meningkatkan kemampuan sistem dalam memahami objek pada gambar.

Melalui project ini dapat dipahami bagaimana teknologi:
- Computer Vision
- CNN
- Dan Deep Learning
diimplementasikan pada pengolahan citra digital secara nyata.

# 2. Kelebihan Sistem

Kelebihan sistem:
- Implementasi sederhana dan mudah digunakan
- Mampu melakukan object detection otomatis
- Mampu melakukan image classification
- Menghasilkan visualisasi detection yang jelas
- Dan dapat digunakan sebagai media pembelajaran Computer Vision.

# 3. Kekurangan Sistem

Walaupun sistem berhasil diimplementasikan, masih terdapat beberapa keterbatasan, antara lain:
- Masih menggunakan pretrained model
- Belum menggunakan custom dataset
- Beberapa objek masih mengalami salah detection
- Dan hasil detection dipengaruhi kualitas gambar

# 4. Saran Pengembangan

Beberapa pengembangan yang dapat dilakukan pada project ini:
- Menggunakan custom dataset
- Melakukan training model sendiri
- Menambahkan realtime webcam detection
- Menambahkan tracking object
- Dan mengembangkan sistem berbasis video realtime

Selain itu sistem juga dapat dikembangkan menjadi:
- Smart surveillance
- Traffic monitoring
- Face recognition
- Dan AI realtime monitoring system

# 5. Penutup Akhir

Project ini berhasil menjadi implementasi sederhana Deep Learning berbasis CNN menggunakan YOLOv8 untuk object detection dan classification.

Melalui project ini diperoleh pemahaman mengenai:
- Cara kerja Computer Vision
- Implementasi CNN
- Object detection
- Image classification
- Dan pengolahan citra digital menggunakan AI modern

Project ini juga dapat menjadi media pembelajaran dan portfolio implementasi Artificial Intelligence berbasis Computer Vision.
